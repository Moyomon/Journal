# Journal - ゲーム翻訳ツール

ゲーム画面の指定した範囲を、DeepLやGemini AIを使って翻訳するツールです。

## ✨ このツールの特徴
*1. 最新のAI翻訳に対応
DeepL / Gemini AI 連携: 自然な日本語でストーリーを楽しめます。（※利用には各APIキーの設定が必要です）
精度の選択: 翻訳エンジンを選べるので、文脈に合わせた最適な翻訳が可能です。
*2. ゲームプレイを邪魔しない操作感
ワンタッチ翻訳: キーボードのファンクションキーや、ゲームパッドのスティック押し込みなどで、プレイ中にサッと翻訳範囲を指定できます。
自由なカスタマイズ: 翻訳結果の文字サイズを変更可能。モニターの大きさや好みに合わせて調整できます。
*3. ストーリーの振り返りも簡単
翻訳ログ機能: 直近10件までの翻訳結果を保存。
「今のセリフ、読み飛ばしちゃった！」という時でも、ログからストーリーを簡単に見返せます。
