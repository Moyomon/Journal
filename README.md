# Journal - ゲーム翻訳ツール

ゲーム画面の指定範囲を翻訳し、ストーリーの把握をサポートするツールです。

## 主な機能

*   **高精度なAI翻訳**: DeepL翻訳 および Gemini AI に対応。※APIキー設定が必要です。
*   **翻訳結果の文字サイズ変更**: 翻訳結果の文字の大きさを変えられます。
*   **翻訳ログ表示**: 10件までの翻訳結果ログを表示、古いメッセージを読み戻せます。
*   **簡単操作**: ファンクションキーまたはゲームパッドのスティックで指定エリアを翻訳。
